O que é JavaScript?

JavaScript é uma das principais linguagens de desenvolvimento web. É a linguagem responsável pela programação e comportamento dinâmico das páginas.

Aplicações comuns

  • Manipulação de imagens e animações
  • Validação de formulários
  • Mudanças dinâmicas de conteúdo
  • Comunicação com servidores (APIs)
  • Desenvolvimento de aplicações web responsivas

Conceitos Fundamentais

Variáveis em JavaScript

Variáveis são espaços na memória que armazenam valores. Em JavaScript, temos três formas de declarar variáveis:

var

var nome = "João";
// Escopo de função
// Pode ser redeclarada
// Hoisting (elevação)

let

let idade = 25;
// Escopo de bloco
// Não pode ser redeclarada
// Não sofre hoisting

const

const PI = 3.14;
// Escopo de bloco
// Não pode ser redeclarada
// Valor constante

Funções em JavaScript

Funções são blocos de código que executam tarefas específicas e podem ser reutilizados.

Função Tradicional

function somar(a, b) {
  return a + b;
}

// Chamada
somar(5, 3); // 8

Arrow Function

const somar = (a, b) => {
  return a + b;
};

// Versão resumida
const somar = (a, b) => a + b;

Estruturas de Controle

JavaScript oferece várias estruturas para controlar o fluxo de execução do código.

Condicionais

if (condicao) {
  // código se verdadeiro
} else {
  // código se falso
}

// Operador ternário
const resultado = condicao ? "Sim" : "Não";

Loops

// For
for (let i = 0; i < 5; i++) {
  console.log(i);
}

// While
let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

Objetos e Arrays

Objetos e arrays são estruturas de dados fundamentais em JavaScript.

Objetos

const pessoa = {
  nome: "Maria",
  idade: 30,
  profissao: "Desenvolvedora",
  apresentar() {
    return `Olá, sou ${this.nome}`;
  }
};

Arrays

const frutas = ["maçã", "banana", "laranja"];

// Métodos úteis
frutas.push("manga");  // Adiciona ao final
frutas.pop();          // Remove do final
frutas.map(f => f.toUpperCase());

Métodos de Array Importantes

  • map() - Cria um novo array transformando cada elemento
  • filter() - Cria um novo array com elementos que passam em um teste
  • reduce() - Reduz o array a um único valor
  • find() - Encontra o primeiro elemento que satisfaz uma condição
  • sort() - Ordena os elementos do array

DOM: Manipulação de Documentos

O Document Object Model (DOM) é uma interface de programação para documentos HTML. Ele representa a página como uma árvore de objetos, permitindo que JavaScript interaja com elementos HTML.

document
html
head
body
div
p

Seleção de Elementos

// Seleciona por ID
const elemento = document.getElementById("meuId");

// Seleciona por classe
const elementos = document.getElementsByClassName("minhaClasse");

// Seletores modernos
const elemento = document.querySelector("#meuId");
const elementos = document.querySelectorAll(".minhaClasse");

Manipulação de Elementos

// Modificar conteúdo
elemento.innerHTML = "Novo conteúdo";
elemento.textContent = "Texto simples";

// Modificar atributos
elemento.setAttribute("src", "imagem.jpg");
elemento.classList.add("destaque");

// Criar elementos
const novoElemento = document.createElement("div");

Conceitos Avançados

Programação Assíncrona

JavaScript executa código de forma assíncrona, permitindo que operações demoradas não bloqueiem o resto do programa.

Promises

const minhaPromise = new Promise((resolve, reject) => {
  // Operação assíncrona
  if (sucesso) {
    resolve(resultado);
  } else {
    reject(erro);
  }
});

minhaPromise
  .then(resultado => console.log(resultado))
  .catch(erro => console.error(erro));

Async/Await

async function buscarDados() {
  try {
    const resposta = await fetch('https://api.exemplo.com/dados');
    const dados = await resposta.json();
    return dados;
  } catch (erro) {
    console.error('Erro:', erro);
  }
}

Armazenamento no Navegador

JavaScript oferece diversas formas de armazenar dados no navegador do usuário.

localStorage

  • Persistente até ser explicitamente removido
  • Armazena strings (usar JSON para objetos)
  • Limite de ~5MB
  • Não expira
// Salvar dados
localStorage.setItem("usuario", "João");

// Recuperar dados
const usuario = localStorage.getItem("usuario");

// Remover dados
localStorage.removeItem("usuario");

sessionStorage

  • Mantém dados apenas durante a sessão
  • Limpo quando o navegador é fechado
  • Mesma API que localStorage
  • Útil para dados temporários
// Mesma API que localStorage
sessionStorage.setItem("tempData", "valor");
const tempData = sessionStorage.getItem("tempData");

Consumo de APIs

JavaScript permite a comunicação com servidores para buscar ou enviar dados através de APIs.

Fetch API

// GET request
fetch('https://api.exemplo.com/dados')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Erro:', error));

// POST request
fetch('https://api.exemplo.com/dados', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({ nome: 'João' })
})

Axios (biblioteca)

// GET request
axios.get('https://api.exemplo.com/dados')
  .then(response => console.log(response.data))
  .catch(error => console.error('Erro:', error));

// POST request
axios.post('https://api.exemplo.com/dados', {
  nome: 'João'
})
